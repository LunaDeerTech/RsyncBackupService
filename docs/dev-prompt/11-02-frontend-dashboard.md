# 11-02 前端仪表盘页面

## 前序任务简报

仪表盘后端 API 已完成：5 个接口返回总览卡片数据、风险事件列表、备份趋势（7 天每日成功/失败）、重点关注实例（容灾率最低的 5-8 个）和即将执行的任务列表。

## 当前任务目标

实现管理员仪表盘页面（`/dashboard`），展示系统全局状态总览、风险预警、备份趋势和重点关注实例。

## 实现指导

### 1. 路由与页面

- 路由：`/dashboard`，admin 权限
- 页面组件：`pages/dashboard/DashboardPage.vue`
- admin 登录后的默认首页

### 2. API 模块

```typescript
// api/dashboard.ts
function getOverview(): Promise<DashboardOverview>
function getRisks(params?: PaginationParams): Promise<PaginatedData<RiskEvent>>
function getTrends(): Promise<DashboardTrends>
function getFocusInstances(): Promise<FocusInstance[]>
function getUpcomingTasks(): Promise<UpcomingTask[]>
```

### 3. 页面布局

使用 CSS Grid 或 Flexbox 组织多个区域，建议布局：

```
┌─────────────────────────────────────────────────┐
│              总览卡片区（5 个卡片横向排列）         │
├──────────────────────┬──────────────────────────┤
│   运行态与风险区      │    趋势与分布区            │
│   - 当前任务/队列     │    - 备份结果趋势图         │
│   - 风险提醒列表      │    - 实例健康分布           │
│                      │    - 即将执行的任务         │
├──────────────────────┴──────────────────────────┤
│              重点关注实例区                        │
│    容灾率最低的实例卡片 × N，点击跳转详情           │
├─────────────────────────────────────────────────┤
│              快捷入口区                           │
└─────────────────────────────────────────────────┘
```

### 4. 总览卡片区

5 个 AppCard 横向排列（响应式：移动端 2 列或纵向排列）：
- **运行中任务**：数字 + 图标，点击可展开任务列表
- **异常实例**：数字 + 图标（warning 色），点击跳转实例列表
- **待处理风险**：数字 + 图标（error 色），点击展开风险区
- **系统容灾率**：分数 + 等级标签（颜色跟随等级）
- **目标健康度**：healthy/degraded/unreachable 数字，用小型 badge 或色块展示

### 5. 运行态与风险区

**当前任务列表**：
- 运行中 + 排队中的任务表格
- 列：实例名称、任务类型、状态、进度（后续阶段十三增强为实时进度）
- 无任务时显示「暂无运行中任务」

**风险提醒列表**：
- 未解决风险事件列表
- 列：严重程度（AppBadge）、来源类型、关联实例/目标、描述、时间
- 最多显示前 10 条，「查看全部」链接跳转到风险事件页面

### 6. 趋势与分布区

**备份结果趋势图**：
- 最近 7 天的柱状图或折线图
- 用简单 CSS 实现柱状图（无需引入图表库）
- 每天两列：绿色（成功）+ 红色（失败）
- 底部显示日期标签

**实例健康分布**：
- 饼图或横向堆叠条形图
- 4 种等级对应颜色：safe/caution/risk/danger
- 显示各等级实例数量

**即将执行的任务**：
- 列表展示未来 24 小时内的计划任务
- 列：实例名称、策略名称、类型、预计执行时间（相对时间如"2 小时后"）

### 7. 重点关注实例区

- 横向卡片列表，每个卡片展示：
  - 实例名称
  - 容灾率分数 + 等级色标
  - 未解决风险数
  - 上次备份状态和时间
- 点击卡片跳转到实例详情页
- 使用 overflow-x 滚动或换行排列

### 8. 快捷入口区

- 4 个小卡片/按钮：实例列表、备份目标、远程配置、审计日志
- 带图标 + 文字，点击跳转对应页面

### 9. 数据刷新

- 进入页面时加载所有数据
- 后台每 30 秒自动刷新（使用 `setInterval` + `onUnmounted` 清理）
- 刷新时不显示全局 loading，静默更新数据

### 10. 响应式

- 桌面端（≥1024px）：多列布局
- 平板（768-1024px）：2 列
- 移动端（<768px）：单列堆叠

## 验收目标

1. admin 登录后默认进入仪表盘页面
2. 总览卡片正确显示运行中任务数、异常实例数、风险数、系统容灾率
3. 备份趋势图展示最近 7 天的成功/失败分布
4. 风险列表展示未解决的风险事件
5. 重点关注实例按容灾率排序展示
6. 即将执行的任务列表正确显示
7. 页面每 30 秒自动刷新数据
8. 移动端响应式布局正常
9. 深色/浅色主题下各区域样式正确
